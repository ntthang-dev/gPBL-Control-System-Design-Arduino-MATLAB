# How to Contribute

Thank you for your interest in this project!

## Academic Project Status

This repository contains work for an academic project as part of the gPBL (global Project Based-Learning) program between SIT, HCMUT, HUST, ITB, and TNI.

As such, **we are not actively seeking external contributions** or bug reports at this time.

## Questions or Suggestions

If you are a fellow student, faculty member, or TA involved in the gPBL program, please feel free to reach out to the author directly via email or the program's official communication channels.
